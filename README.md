## Quick start

1. Clone this repo using `git clone`
2. Run `npm install` to install dependencies. 
3. Run `npm start` and open your browser at http://localhost:3000  

 
### Scripts location  


+ old ==> new

+ auth.js ==> src/containers/auth.jsx
+ feed.js ==> src/containers/feed.jsx
